🚀 Dev Playground

A collection of small projects and mini-games I built while learning Python and programming fundamentals.
This project is designed as an interactive web-based playground that recreates beginner Python concepts using HTML, CSS, and JavaScript.

🌟 Features
🧮 Calculator with:
Arithmetic operations
Calculation history
Keyboard support
✊ Rock Paper Scissors
Play against the computer
Live score tracking
📐 Factorial Calculator
Recursive factorial calculation
Large number handling
🎯 Number Guessing Game
Random number generation
Attempt counter
Hint system
📋 Contacts Manager
Add/Delete contacts
Search functionality
🧾 Bill Calculator
Itemized billing system
Dynamic total calculation
🔢 FizzBuzz Visualizer
Interactive FizzBuzz grid
Live statistics
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)

🎯 Purpose of This Project

This repository is basically my learning journey.
Instead of only writing Python programs in the terminal, I wanted to turn beginner programming concepts into interactive applications with a modern UI.
